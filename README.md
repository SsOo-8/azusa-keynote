Azusa
=============

Azusaは、大体いい感じになるKeynoteテンプレートです。

### 解説記事

<http://memo.sanographix.net/post/82160791768>

### 動作環境

- Mac OSX Mavericks 以降

## Install

- `azusa.zip` を解凍
- `azusa.key` を Keynote で開く
- ファイル -> テーマを保存 でテーマとして登録できる
